Emmet in Visual Studio Code

Support for Emmet snippets and expansion is built right into Visual Studio Code, no extension required. Emmet 2.0 has support for the majority of the Emmet Actions including expanding Emmet abbreviations and snippets.

EMMETS
Symbol Definition
! Expands to the complete html5 template
tag name Expand the complete the tag
> Used to create child elements within a parent element
+ Used to create sibling elements
^ Moves up one level in the hierarchy, allowing you to add sibling elements to parent elements.
* Create multiple elements
